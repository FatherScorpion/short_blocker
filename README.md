# YouTube Shorts 時間制限

YouTube Shortsに設定可能な時間制限をかけるChrome拡張機能です。制限時間に達すると、3段階の確認ポップアップを通過するまで視聴できなくなります。

## 機能

- **時間制限の設定**: 5分〜120分、またはカスタムで1日あたりの視聴時間を設定
- **制限到達時**: オーバーレイが表示され、動画を一時停止
- **視聴再開**: 3段階のくどい確認ポップアップをすべて「続ける」で通過すると視聴再開可能
- **日次リセット**: 毎日0時（日本時間）に使用時間がリセット

## インストール方法（開発者モード）

1. このフォルダをダウンロードまたはクローン

2. **アイコンの作成**（初回のみ）:
   - `icons` フォルダを作成
   - 16x16、48x48、128x128 のPNGアイコンを `icons/icon16.png`、`icons/icon48.png`、`icons/icon128.png` として配置
   - アイコンがない場合、Chromeはデフォルトアイコンで表示されます（開発時は `manifest.json` の `action` と `icons` を一時的に削除しても動作します）

3. Chromeで `chrome://extensions/` を開く

4. 右上の「デベロッパーモード」をONにする

5. 「パッケージ化されていない拡張機能を読み込む」をクリック

6. このフォルダ（`short_blocker`）を選択

7. 拡張機能がインストールされます

## 使い方

1. 拡張機能アイコンをクリックして制限時間を設定（例: 30分）
2. YouTube Shortsを視聴
3. 制限時間に達するとオーバーレイが表示されます
4. 視聴を続けたい場合は、3つの確認を順に「続ける」で通過
5. やめる場合は「やめる」を選択（オーバーレイが閉じ、再度アクセス時に最初から）

## ファイル構成

```
short_blocker/
├── manifest.json          # 拡張機能の設定
├── background.js           # 時間計測・ストレージ管理
├── content.js             # YouTubeページへの注入・オーバーレイ表示
├── popup.html             # 設定画面
├── popup.js               # 設定のロジック
├── popup.css              # ポップアップのスタイル
├── icons/                 # アイコン（16, 48, 128px）
└── README.md              # このファイル
```

## 技術仕様

- **Manifest V3** 対応
- **chrome.storage.local** で設定を保存（オフライン動作、データはブラウザ内のみ）
- **yt-navigate-finish** イベントでYouTubeのSPAナビゲーションを検知
